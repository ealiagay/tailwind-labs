ESTILOS: Tipografía, Jerarquía y Estados

1. Tamaños de Letra (text-xs a text-4xl)

En Tailwind no usamos píxeles fijos. Usamos esta escala (basada en rems) para que el texto se adapte a las pantallas de todos los usuarios.

text-xs: Para etiquetas minúsculas y errores.
text-sm: El estándar para labels de formularios y botones.
text-base: (Por defecto) Para párrafos largos de lectura.
text-lg: Para subtítulos.
text-2xl: Para títulos principales.
