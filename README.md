# Primeros Pasos en Arquitectura Frontend con Tailwind CSS

## 🚀 Cómo ver el proyecto 🚀
Para replicar este entorno de desarrollo y ver los laboratorios en tiempo real, sigue estos pasos:

1. **Descargar las dependencias:**
   `npm install`

2. **Iniciar el proyecto:**
   `npm run dev`

3. **Ver el resultado:** Abre el archivo `src/index.html` en tu navegador para ver el Dashboard final.

---

## 📂 ¿Qué hay en cada laboratorio?

| Archivo | ¿Qué aprendí aquí? | Conceptos Clave |
| :--- | :--- | :--- |
| **`index.html`** | **Proyecto Final: Dashboard** | Es la unión de todo. |
| **`01` al `04`** | **Cimientos de CSS** | Entender el Box Model, cómo fluye el texto y hacer que todo sea responsivo con Flexbox y Grid. |
| **`05-config`** | **Identidad y Colores** | Usa colores propios (`uni-red`, `fac-blue`) y el **Modo Oscuro** automático. |
| **`06-animations`** | **Movimiento y Vida** | Cómo hacer que los botones reaccionen suavemente sin gastar mucha energía de la PC. |

---


## 🛠️ Tecnologías usadas
*   **Tailwind CSS v4** para estilos rápidos y modernos.
*   **Vite** para compilar y ver mis cambios al instante.
*   **Diseño Mobile-First** para que la interfaz se adapte desde celulares hasta monitores grandes.
